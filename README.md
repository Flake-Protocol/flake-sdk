# Flake SDK

The official software development kit for integrating with our Decentralized Exchange. This library provides the essential tools for calculating swap rates, managing liquidity pairs, and interacting with our smart contracts.

## Features
- **Price Math:** Precision calculations for constant product AMMs ($x * y = k$).
- **Multi-chain Support:** Pre-configured constants for Ethereum, Base, and Arbitrum.
- **Type Safety:** Full TypeScript support for reliable DeFi integrations.
- **Contract Interfaces:** Includes standard Solidity interfaces for easy protocol extension.
